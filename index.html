🖶 월 출근부 PDF 출력

사용 방법
  1. 연도/월 선택 후 PDF 미리보기 & 출력 클릭
  2. 새 창에서 출근부 미리보기 확인
  3. Ctrl+P → 용지 방향 가로 → 여백 없음/최소
  4. 대상을 PDF로 저장 선택하면 PDF 파일로 다운로드
  5. 실제 프린터를 선택하면 종이 출력도 가능
⚠ 팝업이 차단될 경우 브라우저 주소창의 팝업 허용 버튼을 클릭하세요.
